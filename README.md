# InfoSphere Site

Static public pages for InfoSphere developer-console URLs.

Suggested GitHub Pages use:

1. Create a public GitHub repo named `infosphere-site`.
2. Push this folder to that repo.
3. Enable GitHub Pages from the `main` branch.
4. Use the published site URL for app website/about/privacy/data-use fields.

No app credentials, OAuth client IDs, tokens, local build files, or user data belong in this repo.

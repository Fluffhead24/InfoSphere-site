InfoSphere
Privacy X data use

Android-first information diet dashboard

Understand where your information comes from.

InfoSphere maps screen-based information time across apps and adds optional platform-native context where the user authorizes it.

What InfoSphere Uses

The prototype uses user-granted Android Usage Access to estimate app-level information time across weekly, monthly, six-month, one-year, and locally retained all-time windows.

What It Adds

Optional connected accounts can add aggregate platform-native signals, such as YouTube channel taxonomy, X followed-account metadata summaries, and Reddit Devvit public community context.

What It Does Not Do

InfoSphere does not read messages, scrape feeds, inspect posts or videos, capture network traffic, collect social passwords, or sell user data.
